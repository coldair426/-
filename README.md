# 개요

- 프로젝트명 : 호사로운
- 기간 : 2023.01.05. ~ 2023.01.30.
- 목표 : 펜션 호사로운의 효과적인 정보 제공과 모객 효과를 줄 수 있는 페이지
- 동기 : 기존에 펜션 호사로운에 대한 정보가 여러 플랫폼에 흩어져 있었다. 이 정보들을 모으고 안내할 페이지가 필요했다. 또한 공부했던 HTML, CSS, JS 지식을 실제 페이지 제작에서 응용해 보고 싶었다.
